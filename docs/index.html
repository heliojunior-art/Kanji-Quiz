📋 Sobre o Projeto

O Kanji Quiz é uma plataforma gamificada de alto desempenho projetada especificamente para auxiliar estudantes da língua japonesa a memorizarem de forma ativa os 1.447 Kanjis oficiais do exame internacional JLPT (N5 ao N1). O sistema resolve com rigor os requisitos de usabilidade e responsividade exigidos para uma aplicação moderna.

🛠️ Diferenciais Técnicos Atendidos

✨ Persistência e Integridade de Dados Back-end

Estrutura Django com SQLite utilizando o decorador @transaction.atomic para gravação de respostas sem riscos de corrupção ou travamentos de memória na nuvem.

📱 Design Gamificado de Alta Performance Front-end

Interface responsiva construída em Bootstrap v5. Otimizada para celulares e computadores com animações fluidas processadas diretamente via hardware (CSS puro).

🔒 Segurança de Navegação Avançada JavaScript

Implementação de travas de histórico via rotinas de popstate que interceptam ações involuntárias no botão 'Voltar' do navegador, garantindo a proteção do progresso do quiz.

🗂️ Isolamento e Privacidade de Sessão Cookies

Lógica baseada na API request.session do Django. O histórico de acertos é estritamente privado por navegador e conta com destruição automática de dados ao fechar o app.
