📋 Sobre o Projeto

O Kanji Quiz é uma plataforma gamificada de alto desempenho projetada para auxiliar estudantes da língua japonesa a dominarem os 1.447 Kanjis oficiais exigidos no exame internacional JLPT. O projeto atende com rigor a todos os requisitos funcionais e não-funcionais estabelecidos no escopo inicial de desenvolvimento.

🛠️ Diferenciais Técnicos Atendidos

  • ✔ Estrutura Django com SQLite: Persistência de dados íntegra utilizando transações atômicas para salvar sessões e respostas.
  • ✔ Design Otimizado: Interface construída sob demanda usando Bootstrap v5 para garantir usabilidade rápida em computadores e celulares.
  • ✔ Segurança de Fluxo de Jogo: Filtros JavaScript integrados que interceptam o botão voltar do navegador, impedindo perdas acidentais de progresso.
  • ✔ Mecanismo de Dicas em Tempo Real: Modais customizados em CSS/JS puro livres de quebras que oferecem suporte focado ao estudante.
  • ✔ Histórico de Desempenho Inteligente: Painel de estatísticas isolado por sessões privadas de navegação e com autolimpeza automática ao fechar o app.
